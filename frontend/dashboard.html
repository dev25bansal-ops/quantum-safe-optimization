QAOA - Quantum Approximate Optimization Algorithm

Solve combinatorial optimization problems with hybrid quantum-classical variational circuits.

🎯

What is QAOA?

QAOA is a hybrid quantum-classical algorithm designed to find approximate solutions to combinatorial optimization problems. It uses parameterized quantum circuits alternating between cost and mixer Hamiltonians.

📊 Supported Problems

✅ Max-Cut Popular

Partition graph vertices to maximize edges between partitions.

Complexity: NP-hard Applications: Network design, VLSI layout Qubits: n (one per vertex)
✅ Traveling Salesman (TSP) Classic

Find the shortest route visiting all cities exactly once.

Complexity: NP-hard Applications: Logistics, routing Qubits: n² (n cities)
✅ Graph Coloring Useful

Color vertices so no adjacent vertices share the same color.

Complexity: NP-complete Applications: Scheduling, register allocation Qubits: n × k (n vertices, k colors)
✅ Portfolio Optimization Finance

Select assets to maximize return while minimizing risk.

Type: Quadratic optimization Applications: Investment management Qubits: n (one per asset)
✅ Custom QUBO Flexible

Define your own Quadratic Unconstrained Binary Optimization.

Format: Upper triangular matrix Applications: Any binary optimization Qubits: n (number of variables)

⚙️ Algorithm Parameters

Layers (p)
Number of QAOA layers. More layers = better approximation but longer runtime.
Range: 1-20 | Default: 2
Optimizer
Classical optimizer for parameter tuning.
COBYLA, BFGS, ADAM, SPSA, Nelder-Mead, SLSQP
Shots
Number of circuit measurements per evaluation.
Range: 100-100,000 | Default: 1,000
Initial Parameters
Starting values for γ (gamma) and β (beta) angles.
Random, Warm-start, Custom

VQE - Variational Quantum Eigensolver

Find ground state energies for molecular and material simulations using quantum chemistry.

⚛️

What is VQE?

VQE is a hybrid algorithm for finding the lowest eigenvalue of a Hamiltonian, crucial for quantum chemistry simulations. It uses parameterized quantum circuits (ansätze) to prepare trial wavefunctions.

🧪 Supported Molecules

H₂
Hydrogen

Diatomic hydrogen molecule

Qubits: 2-4
Bond length: 0.74 Å
Ground state: -1.137 Ha
LiH
Lithium Hydride

Alkali metal hydride

Qubits: 4-12
Bond length: 1.60 Å
Ground state: -7.98 Ha
HeH⁺
Helium Hydride

Helium hydride cation

Qubits: 2-4
Bond length: 0.77 Å
Ground state: -2.86 Ha
H₂O
Water

Water molecule (advanced)

Qubits: 8-14
Geometry: 104.5° angle
Ground state: -76.4 Ha

📐 Ansatz Types

UCCSD
Chemistry

Unitary Coupled Cluster Singles and Doubles - chemically inspired ansatz with high accuracy.

Parameters: O(n⁴) Accuracy: High Depth: Deep
Hardware Efficient
Hardware

Native gate ansatz optimized for specific quantum hardware topology.

Parameters: O(n × layers) Accuracy: Medium Depth: Shallow
SU(2)
General

Special unitary group rotations with entanglement layers.

Parameters: 3n × layers Accuracy: Medium Depth: Configurable
RY-RZ
Hardware

Alternating RY and RZ rotation layers with CZ entanglement.

Parameters: 2n × layers Accuracy: Medium Depth: Shallow

🔬 Hamiltonian Types

Molecular Hamiltonian

Second-quantized electronic structure Hamiltonian using Jordan-Wigner or Bravyi-Kitaev mapping.

Ising Model

Spin-1/2 lattice model: H = -J Σᵢⱼ σᵢᶻσⱼᶻ - h Σᵢ σᵢˣ

Heisenberg Model

Quantum spin model: H = J Σᵢⱼ (σᵢˣσⱼˣ + σᵢʸσⱼʸ + σᵢᶻσⱼᶻ)

Custom Pauli

Define arbitrary Pauli string Hamiltonians with custom coefficients.

Create a new VQE job for molecular ground state calculation.

Example: H₂ Ground State
{
  "molecule": "h2",
  "bond_length": 0.74,
  "ansatz_type": "UCCSD",
  "optimizer": "BFGS"
}

Quantum Annealing

Native quantum annealing for QUBO and Ising model problems using adiabatic quantum computing.

⚡

What is Quantum Annealing?

Quantum annealing exploits quantum tunneling to find the global minimum of an optimization problem. The system starts in a superposition and slowly evolves toward the ground state of the problem Hamiltonian.

✨ Features

🔢
5000+ Qubits

Access to D-Wave's latest Advantage processors with 5000+ qubits.

🔗
Automatic Embedding

Automatic minor embedding for Pegasus and Zephyr topologies.

🔄
Hybrid Solvers

Combine quantum and classical resources for larger problems.

⚙️
Chain Optimization

Automatic chain strength tuning for improved solution quality.

📝 Problem Formulations

QUBO (Quadratic Unconstrained Binary Optimization)

min xTQx where x ∈ {0,1}n

Q = [[−1, 2, 0],
     [ 0,−1, 2],
     [ 0, 0,−1]]
Ising Model

H = Σᵢ hᵢsᵢ + Σᵢⱼ Jᵢⱼsᵢsⱼ where s ∈ {-1,+1}n

h = [0.5, -0.5, 0.5]
J = {(0,1): -1, (1,2): -1}

⏱️ Annealing Parameters

Annealing Time
Total time for the quantum annealing process.
Range: 1-2000 μs | Default: 20 μs
Number of Reads
Number of independent annealing runs.
Range: 1-10,000 | Default: 1,000
Annealing Schedule
Control the evolution of transverse field.
Linear, Pause, Quench, Custom
Chain Strength
Coupling strength for embedded chains.
Auto-tuned or manual: 0.1-10.0
