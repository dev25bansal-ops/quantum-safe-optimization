## GitHub Desktop Steps

### 1. Open GitHub Desktop
- Press **Windows key** and search for "GitHub Desktop"
- Or run: `github-desktop`

### 2. Add Repository
- Click **File** → **Add Local Repository**
- Navigate to: **D:\Quantum**
- Click **Add Repository**

### 3. Publish to GitHub
- Click **Publish repository** button (top right corner)
- Fill in:
  - **Name:** `quantum-safe-optimization`
  - **Description:** `Quantum-Safe Secure Optimization Platform - Research platform for IEEE Quantum Week 2025`

- Click **Publish repository**

- GitHub Desktop handles large uploads automatically
- It will retry if connection drops
- Do not close GitHub Desktop during upload
